# <img src="./assets/images/money.png" width="40" height="40" alt="Seiker Finance Icon" valign="middle"> Seiker Finance

O **Seiker Finance** é um ecossistema de gestão financeira pessoal desenvolvido para oferecer alta performance e uma experiência de usuário imersiva. O app combina organização rigorosa com uma interface moderna e fluida.

---

## 🚀 Funcionalidades Principais

* **Gestão de Orçamento Mensal**: Controle de renda e despesas segmentadas por categorias customizáveis.
* **Sistema de Parcelamento**: Distribuição automática de gastos em meses futuros com controle de progresso (ex: 1/12).
* **Dashboard de Analytics**: Visualização clara através de gráficos de pizza (distribuição) e barras (Renda x Gastos).
* **Tema Dinâmico**: Alternância manual entre os modos **Dark** e **Light** com persistência de dados.
* **Experiência de Navegação**: Sistema de abas (Tabs) e suporte a gestos de *swipe* para troca rápida de meses.
* **Autenticação Integrada**: Fluxo de Login e Cadastro robusto utilizando Firebase Authentication.

---

## 🛠️ Tecnologias Utilizadas

* **Framework**: [React Native](https://reactnative.dev/) com [Expo SDK](https://expo.dev/).
* **Linguagem**: [TypeScript](https://www.typescriptlang.org/) para maior segurança e tipagem estrita.
* **Backend**: [Firebase Firestore](https://firebase.google.com/docs/firestore) (Banco de dados em tempo real).
* **Animações & Gestos**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) e [Gesture Handler](https://docs.swmansion.com/react-native-gesture-handler/).
* **Gráficos**: [React Native Gifted Charts](https://github.com/Abhinandan-Kushwaha/react-native-gifted-charts).
* **Ícones**: [Lucide React Native](https://lucide.dev/).

## ⚙️ Instalação

1. Clone o repositório:
```bash
git clone [https://github.com/seu-usuario/seiker-finance.git](https://github.com/seu-usuario/seiker-finance.git)
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor:
```bash
npx expo start -c
```
